# Modelos constitutivos

Implementação de alguns modelos constitutivos e classes básicas.

## Testes

Os seguintes scripts estão prontos para testar com o modelo de Cam-Clay:

- O arquivo [run.py](./run.py) tem um caso simplificado
- O arquivo [app.py](./app.py) cria um aplicativo com interface no streamlit

Para instalar as dependências, recomenda-se o uso do
[`uv`](https://docs.astral.sh/uv/).
